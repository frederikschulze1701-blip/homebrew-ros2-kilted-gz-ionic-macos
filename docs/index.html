ROS 2 Kilted + Gazebo Ionic

Pre-compiled Homebrew tap for macOS Apple Silicon

Apple Silicon macOS ROS 2 Kilted Gazebo Ionic arm64

Quick Install

Two commands and you're ready to go:

# 1. Add the tap
brew tap frederikschulze1701-blip/ros2-kilted-gz-ionic-macos

# 2. Install ROS 2 + Gazebo
brew install ros2-kilted-gazebo-ionic

Setup

Source the setup script in your terminal:

source $(brew --prefix)/opt/ros2-kilted-gazebo-ionic/setup.zsh

To load it automatically, add it to your shell profile:

echo 'source $(brew --prefix)/opt/ros2-kilted-gazebo-ionic/setup.zsh' >> ~/.zshrc

What's Included

  • ROS 2 Kilted — the latest ROS 2 release
  • Gazebo Ionic — next-gen robotics simulator
  • Pre-compiled for macOS Apple Silicon (arm64)
  • No source build required — install in minutes, not hours

Why This Tap?

Building ROS 2 from source on macOS can take hours and often fails due to dependency issues. This tap provides a pre-compiled distribution that installs cleanly via Homebrew — the way macOS packages should work.
